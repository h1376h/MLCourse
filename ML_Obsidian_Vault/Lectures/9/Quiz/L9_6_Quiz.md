# Lecture 9.6: Sampling Techniques and Strategies Quiz

## Overview
This quiz contains 25 comprehensive questions covering sampling techniques, including random sampling, stratified sampling, systematic sampling, cluster sampling, and practical applications. All questions are designed to be solvable using pen and paper with concrete examples and calculations.

## Question 1

### Problem Statement
You have a dataset with $1000$ customers and want to select a sample of $100$ for a survey.

#### Task
1. If you use simple random sampling, what's the probability that any specific customer is selected?
2. What's the probability that a customer is NOT selected?
3. What's the expected number of customers from the first $200$ who will be selected?
4. What's the main advantage of simple random sampling?
5. What's the main disadvantage of simple random sampling?

For a detailed explanation of this question, see [Question 1: Simple Random Sampling](L9_6_1_explanation.md).
